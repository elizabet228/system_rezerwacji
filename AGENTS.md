# Role Mikroserwisów - Roomly

# User-Service
- Zarządzanie kontami użytkowników i bezpieczeństwem.
- Rejestracja, logowanie, hashowanie haseł oraz weryfikacja tokenów JWT.

# Booking-Service
- Zarządzanie logiką biznesową rezerwacji.
- Przechowywanie informacji o salach, sprawdzanie dostępności i proces tworzenia/usuwania rezerwacji.
